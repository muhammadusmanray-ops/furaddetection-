# 📖 User Guide: FraudPro Enterprise

## Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `streamlit run app.py`

## Features
- **Dashboard:** View live metrics and fraud trends.
- **Audit Log:** Search through transaction history.
- **Live Test:** Manually input transaction data to check for fraud.
- **AI Chat:** Talk to the security assistant for fraud analysis.

## Configuration
- Add your `GROQ_API_KEY` to `.streamlit/secrets.toml` for AI functionality.
